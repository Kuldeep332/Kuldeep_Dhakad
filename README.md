Kuldeep Dhakad – Screening Test (TANDEMLOOP)

This repository contains my solutions for the First Screening Test
for the Full Stack Developer position at TANDEMLOOP.

Language Used:
Java

Files Included:
1. Problem-1.java  – Calculator program using class (supports add, sub, mul, div)
2. Problem-2.java  – Generates odd number series based on user input
3. Problem-3.java  – Generates odd number series based on odd/even logic
4. Problem-4.java  – Counts how many numbers are divisible by 1 to 9 from a given list

How to Run:
- Open the project in any Java IDE or command prompt.
- Compile using: javac Problem-1.java
- Run using: java Problem-1
- Provide inputs from the console as asked.

All programs take dynamic input from the user and are tested locally.
